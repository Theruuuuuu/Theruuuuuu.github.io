在C#中使用ChartJs

TheRU

May 2, 2023

C#, Web

Chart.js??

他是一個javascript的程式庫，能夠實現圖表功能。

使用方法

在你要使用的HTML，加上引入

<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>

然後在你想要的地方加入

<div>
  <canvas id="myChart"></canvas>
</div>

根據官方說明文件建立一個簡單的長條圖

<div>
  <canvas id="myChart"></canvas>
</div>

<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>

<script>
  const ctx = document.getElementById('myChart');

  new Chart(ctx, {
    type: 'bar',
    data: {
      labels: ['Red', 'Blue', 'Yellow', 'Green', 'Purple', 'Orange'], 
      datasets: [{
        label: '# of Votes',
        data: [12, 19, 3, 5, 2, 3],
        borderWidth: 1
      }]
    },
    options: {
      scales: {
        y: {
          beginAtZero: true
        }
      }
    }
  });
</script>
